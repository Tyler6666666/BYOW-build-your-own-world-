# BYOW - Build Your Own World

Windows quick start:

1. Make sure `library-sp26` is next to this project folder, for example:
   `cs61b/library-sp26/algs4.jar`
2. Double-click `Play_BYOW.bat`.

The launcher compiles `proj5/src` into `proj5/out/game` and starts `core.Main`.

If your `library-sp26` folder is somewhere else, set `CS61B_LIBRARY` to that folder before running the launcher.
