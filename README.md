# BYOW - Build Your Own World

Windows quick start:

1. Double-click `Play_BYOW.bat`.

The launcher compiles `proj5/src` into `proj5/out/game` and starts `core.Main`.

The required CS61B library is included in `library-sp26`. If you want to use a different copy, set
`CS61B_LIBRARY` to that folder before running the launcher.
